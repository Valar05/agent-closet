# Agent Closet Steering

Status: Active steering layer
Purpose: Short operational steering for prompt-building and clean worker initialization.

## Repository truth first

Read before writing. Append before replacing. Cross-link before duplicating.

When a durable doctrine already exists, do not recreate it under a new name. Point workers at the canonical file and add only the missing operational rule.

## Self-insertion prompt building

Self-insertion is an allowed prompt-building doctrine when it is used as calibration, not identity cloning.

The prompt should not ask a worker to become the user. It should identify what part of the user's operating pattern is being externalized into the worker:

- recurring judgment pattern
- responsibility carried
- capability amplified
- blind spot covered
- burden intentionally not inherited
- place where the worker should productively disagree

Canonical doctrine: `shared/doctrine/self-insertion-calibration.md`.

## Clean worker bootstrap

When an async or fresh-context worker is unavailable, treat the worker prompt as the bootstrap loader.

A clean worker prompt should carry only the minimum runtime doctrine needed to execute safely:

1. Identity as responsibility, not personality.
2. Source of truth to read first.
3. Task boundary.
4. Output contract.
5. Write policy.
6. Verification rule.
7. Exit condition.

Do not smuggle a whole conversation into the worker. Load doctrine by reference when possible.

## Prompt construction rule

Before writing an agent prompt, answer:

1. What responsibility owns this task?
2. What source must be read before action?
3. What judgment pattern is being externalized?
4. What must the worker refuse or stop on?
5. What artifact proves completion?
6. What independent verification checks the work?

If those answers are missing, the prompt is not ready. It is vibe paste.

## No-op churn guard

A worker is not done because it generated text. It is done when the requested artifact exists, is linked or committed, and the verification condition passes.

If the worker cannot create the artifact directly, it must produce a dispatch packet that a fresh worker can execute without chat archaeology.
