Authority Stack
1. Agent Closet GitHub repo is the source-of-truth doctrine and prompt-injection layer: https://github.com/Valar05/agent-closet
2. Drive Brain is the corpus, index, registry, promotion log, and retrieval layer.
3. Project repos are execution and evidence layers.
4. Conflict rule: Agent Closet supersedes local Drive Brain instructions when conflicts exist. Drive Brain should point agents to Agent Closet, not replace it.
5. Required behavior: Observe Before Advising / Read Before Report. Inspect the relevant artifact before planning, auditing, critiquing, or recommending.

Cold Agent Boot Path
1. Open Agent Closet first: https://github.com/Valar05/agent-closet
2. Read its current doctrine and prompt-injection layer.
3. Return to Drive Brain for retrieval, index lookup, registry status, promotion logs, and corpus search.
4. Use project repos as execution/evidence layers and verify claims against artifacts.
5. When instructions conflict, follow Agent Closet unless the user explicitly overrides it in the current task.

Required Operating Procedure
Before creating a new artifact:
1. Search for an existing artifact.
2. Read relevant doctrine.
3. Read related project registries.
4. Verify the artifact does not already exist.
5. Create or update the smallest useful asset.
6. Cross-link related assets.
7. Save.
8. Read.
9. Verify.
