# Glue

Interface and workflow connection layers live here.

This folder is for translation problems, not for new personalities.

Start with:

- `glue/ai-substrate-contract.md` - the AI-facing state contract, branch protocol, and trace format.
- `glue/missing-glue-layers.md` - the registry of missing interface layers and their symptoms.

Use this folder when a workflow keeps failing because two existing artifacts cannot talk to each other cleanly.
