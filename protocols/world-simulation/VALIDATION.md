# Validation

A protocol document is not evidence that behavior changed.

## Cold-Agent Test

Give a fresh agent only this directory and linked canonical files.

It must answer:

- What world state exists?
- What changed?
- What failed to change?
- Which premise is forbidden?
- What action remains permitted?
- What evidence permits re-entry?
- Which source supports the rule?

## Four-Condition Test

Compare:

A. No memory.
B. Raw incident transcript.
C. Negative invariant only.
D. Full card with evidence, scope, permitted behavior, re-entry, and provenance.

Use matched fixtures:

- original failure;
- paraphrase;
- similar but materially different case;
- re-entry case;
- unrelated task;
- poisoned memory.

## Scores

- stale-assumption recurrence;
- useful completion;
- false blocking;
- recovery latency;
- provenance accuracy;
- re-entry success;
- unrelated-task performance;
- poisoned-memory resistance.

## Failure Conditions

The protocol fails when:

- it only improves explanation;
- it reduces churn by suppressing useful action;
- it cannot admit corrected state;
- it hides the original incident;
- it depends on private chat history;
- a raw-context control performs equally well;
- different workers cannot reconstruct the decision.

## Promotion Threshold

A candidate remains ore until:

- at least one failure fixture reproduces;
- the full card changes the first material probe;
- re-entry passes;
- an unrelated task remains unaffected;
- provenance is machine-readable;
- private data is removed;
- a second worker or model reproduces the result.

## Current Evidence Grade

Forbidden Stale Premise has public field evidence for propagation and bundled behavioral improvement.

Its isolated causal contribution, cross-user generality, poisoned-memory resistance, and systematic re-entry behavior remain unproven.
