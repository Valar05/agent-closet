# Canonical Fiction Corpus Map

Status: v2 / mining gate
Created: 2026-06-24
Depends on: `corpus/fiction/fiction-manifest.md`

## Purpose

This file decides which version of each fiction work is the current mining target.

The manifest records everything found.
This map chooses the canonical text to read first.

No style, action, pacing, or voice conclusions should be made from older alternates until the canonical candidate is mined.

## Selection Rule

Prefer, in order:

1. Google Doc over DOCX/PDF/TXT export.
2. Revised or v2/v3 version over older draft.
3. Multi-part revised sequence over original single draft.
4. User-confirmed canonical version over inferred version.
5. If uncertain, mark `NEEDS VERIFY` and do not discard alternates.
