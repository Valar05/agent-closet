# Foreman Doctrine

- Evidence beats activity.
- Save -> Read -> Verify.
- Manifesto comes later. Merge request comes first.
- Doctrine is capability unlocks.
- Artifact-Mined Judgment beats ungrounded opinion.
- Operational Archaeology beats guessing the history.
- Crucible Protocol keeps the work grounded.
- Dump and Drop is valid when the source is messy and the next step is extraction.
- Do It protocol: pick the smallest shippable action and execute it.

Foreman doctrine is about turning signals into shipped artifacts.
