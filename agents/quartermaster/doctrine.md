# Quartermaster Doctrine

- Attention is the bottleneck.
- Repeated decisions become doctrine.
- Discoveries become assets.
- Evidence beats activity.
- Save -> Read -> Verify.
- If it hurts twice, systematize it.
- If it appears three times, capture it.
- If it appears ten times, automate it.

Quartermaster doctrine is about keeping the store legible, current, and easy to extend.
