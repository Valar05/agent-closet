# What The Hell Is Agent Closet?

Agent Closet is not a prompt library.

It is not a chatbot personality collection.

Its agent names mark responsibilities, not roleplay. See `shared/doctrine/perspective-guided-command.md`.

It is not a note dump.

Agent Closet is a versioned store for preserving useful behavior.

The central idea is simple:

Discovery -> Doctrine -> Future Behavior

Most useful discoveries are forgotten.
Agent Closet exists to reduce rediscovery.

## What Lives Here

- recurring decisions
- operating rules
- agent orientation packs
- capability doctrines
- artificial continuity artifacts
- patterns that survived reality

## How To Use It

Humans:
1. Read the relevant doctrine.
2. Apply it.
3. Improve it when reality disagrees.
4. Commit the update.

Agents:
1. Orient.
2. Read.
3. Inherit useful behavior.
4. Leave the repository better than you found it.

## What We Learned

The strongest discoveries often became infrastructure.

Nobody spreads an idea.
They spread a habit.

The moment an idea becomes 'obviously', it has become infrastructure.

## Artificial Continuity

The goal is not preserving every conversation.

The goal is preserving the structures that repeatedly generate useful outcomes.

## Final Compression

Conversation
-> Discovery
-> Doctrine
-> Capability
-> Infrastructure

The doctrine is not yours.
The doctrine is not mine.
The doctrine is the artifact left behind by the interaction.
