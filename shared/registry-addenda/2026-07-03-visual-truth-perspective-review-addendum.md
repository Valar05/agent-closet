## Promoted Discoveries

### Visual Truth Authority

When work is visual, the visible acceptance artifact is the authority. Telemetry, DOM state, logs, metrics, cache tokens, source-string tests, green CI, and model confidence are support evidence only.

Cluster routing:
- Animation and Pose Systems
- AI Orchestration
- Project Archaeology

### Visible Relationship Truth

Object presence is not visual proof. Visual QA must compare expected relationship against actual relationship and name the visible mismatch.

Cluster routing:
- Animation and Pose Systems
- AI Orchestration
- Project Archaeology

### Truth Ledger Before Progress Claims

Before calling significant implementation work green, compare repository truth, runtime truth, visual truth, and human truth. If any truth remains red, frame the work as preservation, diagnosis, or blocker.

Cluster routing:
- Project Archaeology
- AI Orchestration
- Animation and Pose Systems
- Software Delivery when applicable

### Perspective-Guided Command Ownership

Perspectives are responsibilities, not personalities. Every significant task needs an owner. Additional perspectives are useful only when they gather evidence or notice failure the owner would miss.

Cluster routing:
- AI Orchestration
- Project Archaeology

### External Critic Code Review

For risky or agent-authored PRs, use an independent critic through read-only repository and PR tools. Verify findings against concrete repository evidence and changed diff lines before reporting or posting.

Cluster routing:
- Software Delivery
- AI Orchestration
- Project Archaeology

### Mechanism-Indexed Learning Curation

Learning maps should index by mechanism, preserve taste signals, avoid low-value retreads, separate documented sources from drama or fiction, and define a return path after detours.

Cluster routing:
- AI Orchestration
- Project Archaeology
- Candidate: Learning Systems / Research Curation

## Duplicate Decisions

- Visual Truth Authority refines red-build evidence, runtime diagnostics, and visual QA doctrine. It does not activate a Visual Production cluster.
- Perspective-Guided Command is ownership and verification. Perspective Graphing remains multi-perspective synthesis.
- External Critic Review belongs under Software Delivery and AI Orchestration. No separate Review Systems cluster.
- Learning Systems / Research Curation remains held until repeated evidence appears outside the current Drive roadmap.

## Archive Guidance

Archive or demote:
- object-presence visual checks without relationship proof;
- telemetry-only visual acceptance claims;
- progress reports hiding a red truth layer;
- pasted source packets substituting for repo/PR tool access;
- recommendation lists that blur documented history, drama, fiction, and current-affairs sludge.
