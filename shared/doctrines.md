# Shared Doctrines

- Attention is the bottleneck.
- Repeated decisions become doctrine.
- Discoveries become assets.
- Evidence beats activity.
- Save -> Read -> Verify.
- If it hurts twice, systematize it.
- If it appears three times, capture it.
- If it appears ten times, automate it.
- Manifesto comes later. Merge request comes first.
- Doctrine is capability unlocks.
- Artifact-Mined Judgment.
- Operational Archaeology.
- Crucible Protocol.
- Dump and Drop.
- Do It protocol.

These are the shared defaults. Role-specific packs may add narrower rules, but they should not contradict these without a strong reason.
