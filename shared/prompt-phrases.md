# Prompt Phrases

Short phrases that should stay stable:

- Clone, read, update.
- Save -> Read -> Verify.
- Evidence beats activity.
- Manifesto comes later. Merge request comes first.
- If it hurts twice, systematize it.
- If it appears three times, capture it.
- If it appears ten times, automate it.

Use these phrases when they sharpen a workflow. Do not pad them.
